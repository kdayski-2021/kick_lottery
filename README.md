# Kick Lottery

Веб-приложение на Vue, которое взаимодействует с контрактами лотереи через `web3`/`web3modal` и отображает текущий раунд, историю раундов и историю ставок клиента.

## Что умеет

- Подключение кошелька через Web3Modal (MewConnect/Frame/Authereum).
- Просмотр текущего раунда:
  - размер текущего джекпота (`Jackpot`)
  - время/блоки до окончания приема ставок (`endBets`)
  - количество участников
- Игра (ставка):
  - ввод количества `Kick`
  - нажатие `Play` отправляет транзакцию `transferAndCall(kickLotteryAddress, amount)` (перед этим вызывается `checkRound`).
- Забрать выигрыш / сменить раунд:
  - `Забрать выигрыш (Round Change)` вызывает `checkRound()`
  - `Force Round Change` вызывает `checkRoundAdmin()` (кнопка отключена, когда раунд “слишком далеко” по блокам)
- История:
  - `История завершенных раундов` (burnedTotal, endDate, jackpot, numwinners)
  - `История ставок клиента` (roundStatus, bet, iswin/prize)

## Стек

- Vue 2 + Vuetify
- web3, web3modal
- Коммуникация с контрактами: чтение по RPC (Infura) + транзакции из кошелька

## Настройка

Параметры сети и адреса контрактов задаются в:

- `src/config/default.json`
  - `networkParameters.rpc` — RPC endpoint (используется для read-only вызовов)
  - `kickLotteryAddress` — адрес контракта лотереи
  - `erc20burnableAddress` — адрес ERC20 burnable контракта

> В `src/api/bia.js` также зашит `infuraId` для провайдеров Web3Modal. Если будете использовать другой Infura проект, при необходимости поменяйте его там.

## Установка

1. Убедитесь, что установлен Node.js и npm.
2. В корне проекта:
   - `npm install`

## Запуск в разработке

- `npm run serve`

## Сборка

- `npm run build`

## Линтинг

- `npm run lint`

## Как пользоваться (коротко)

1. Откройте приложение.
2. Нажмите `Connect Wallet` и подключите сеть/кошелёк.
3. В блоке `Play` укажите количество `Kick` и нажмите `Play`.
4. Для завершения раундов/получения выигрыша используйте кнопки `Round Change` / `Force Round Change`.

## Примечания

- Приложение периодически обновляет данные (в `src/App.vue` опрос выполняется раз в 5 секунд).
- Адрес кошелька отображается в интерфейсе, а все отправки транзакций выполняются из выбранного кошельком провайдера.
